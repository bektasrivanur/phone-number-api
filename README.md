# Telefon Numarası TR'de geçerli mi?

## Kullanılan API
NumberVerify API

## API Endpoint Örneği
http://apilayer.net/api/validate?access_key=${API_KEY}&number=${number}&country_code=TR&format=1

## Uygulama Nasıl Çalışıyor?
- Kullanıcıdan telefon numarası alınır.
- Butona basıldığında API’ye fetch ile istek atılır.
- Gelen veride numaranın TR'de geçerli olup olmadığı kontrol edilir.
- Sonuçlar ekranda kullanıcıya gösterilir.

## Kullanılan JavaScript Konuları
- DOM Manipülasyonu
- Fetch API
- Async / Await
- Fonksiyonlar
- Parametre ve Return
- If / Else ve Koşul Yapıları
